Posicionamiento CSS

Ejemplos de Relative y Absolute

1. Posicionamiento RELATIVE

Un elemento con posicionamiento relativo se desplaza respecto a su posición normal, pero sigue ocupando su espacio original en el flujo del documento.

Contenedor RELATIVE (padre)

Elemento RELATIVE

Desplazado 20px arriba y 30px a la derecha

Elemento Hermano

Código CSS:

.relative-child {
  position: relative;
  top: 20px;
  left: 30px;
}

2. Posicionamiento ABSOLUTE

Un elemento con posicionamiento absoluto se sale del flujo normal y se posiciona respecto a su contenedor RELATIVE más cercano.

Contenedor RELATIVE (contenedor posicionado)

ABSOLUTE
top: 10px | left: 10px
ABSOLUTE
top: 10px | right: 10px
ABSOLUTE
bottom: 10px | left: 10px
ABSOLUTE
bottom: 10px | right: 10px

Este contenido fluye normalmente dentro del contenedor.

Los elementos ABSOLUTE no lo afectan.

Código CSS:

.absolute-parent {
  position: relative;
}

.absolute-child {
  position: absolute;
}

4. Comparativa: Relative vs Absolute

POSICIONAMIENTO RELATIVE

  • ✓ Mantiene su espacio en el flujo
  • ✓ Se desplaza desde su posición original
  • ✓ Afecta a otros elementos
  • ✓ Usa top, bottom, left, right
  • ✓ Útil para ajustes menores

POSICIONAMIENTO ABSOLUTE

  • ✓ Sale del flujo del documento
  • ✓ Se posiciona respecto a su contexto
  • ✓ No afecta a otros elementos
  • ✓ Usa top, bottom, left, right
  • ✓ Ideal para overlays y badges
